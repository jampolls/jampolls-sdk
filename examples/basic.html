Jampolls Embed Examples

Replace YOUR_EMBED_KEY with a key from dashboard → Embeds. The same key renders a poll, rating, or survey depending on which tool is active.

1 · Auto-embed (data attribute)

<script src="https://cdn.jampolls.com/jampolls.min.js"></script>
<div data-jampolls="YOUR_EMBED_KEY"></div>

2 · Manual init

JamPolls.embed('YOUR_EMBED_KEY', '#widget-manual');

3 · Dark theme + callbacks + custom vars

Open DevTools console — onLoad logs tool_type; switch the active tool in Embeds to see onToolChanged.

JamPolls.embed('YOUR_EMBED_KEY', '#widget-dark', {
  theme: 'dark',
  vars: {
    '--jp-primary': '#7c3aed',
    '--jp-radius': '12px',
    'max-width': '480px',
  },
  onLoad: data => console.log('loaded', data.tool_type, data),
  onVote: e  => console.log('poll vote', e),
  onSubmit: e => console.log('rating/survey submit', e),
  onToolChanged: p => console.log('tool switched', p.tool_type),
  onError: e => console.error(e),
});
